# ClipTokk Dashboard

Dashboard connecté pour gérer ClipTokk.com

## Structure

- [[stats]] - Revenue, clients, prospects, posts
- [[accounts]] - Social media accounts (TikTok, Instagram, Threads, YouTube, Twitter)
- [[tools]] - Automation tools status
- [[tasks]] - Critical tasks and priorities
- [[clients]] - Active client list
- [[niches]] - Target niches and markets
- [[weekly]] - Weekly performance metrics

## Accès

- **Web Dashboard**: http://localhost:3333/dashboard.html
- **Telegram Bot**: @cliptokk_sony_bot (à configurer)
- **Obsidian**: Ce vault

## Synchronisation

Tous les fichiers sont synchronisés:
- Modifie dans Obsidian → Dashboard mis à jour
- Utilise ClawBot → Obsidian mis à jour
- Dashboard et bot lisent depuis Obsidian
