## Data Structure

### Big Labels CSV
```
LABEL, PERSONNE, POSTE, LOCALISATION, EMAIL
```
- Major labels: Universal Music Group, Atlantic Records, RCA Records, etc.
- Decision makers: CMO, VP Marketing, Head of Marketing
- Locations: Mostly New York, LA

### Independent Labels CSV
```
name, email, url
```
- 5,500+ independent labels worldwide
- Direct contact emails
- LabelsBase URLs for more info

### YouTubers CSV
```
url, email/0, email/1, email/2, status, error
```
- Gaming/Entertainment YouTubers
- Business emails (management, direct)
- Notable: Adin Ross, Agent00, 72hrs, Ampeterby7

## Priority Segments

### Tier 1 - High Value (Contact First)
- Big Labels VPs and CMOs
- YouTubers with 1M+ subscribers
- Labels with verified emails

### Tier 2 - Medium Value
- Big Labels managers
- YouTubers 100k-1M subscribers
- Active independent labels

### Tier 3 - Volume Play
- All independent labels
- Smaller YouTubers
- Generic label emails (info@)

## Outreach Strategy
1. Personalize for Tier 1 (manual review)
2. Semi-personalized templates for Tier 2
3. Mass outreach for Tier 3 via Instantly.ai
